# Gestión de Incidencias – Spring Boot

Proyecto backend desarrollado en Java con Spring Boot.

Este proyecto es la evolución de una aplicación de consola en Java orientada a objetos,
migrada progresivamente a una API REST siguiendo buenas prácticas de arquitectura.

## Tecnologías
- Java
- Spring Boot
- Maven
